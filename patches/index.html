⚙️ Evaluation Methodology

Baseline Compliance: WCAG 2.2 Level AA

Scanning Tool: axe-core 4.x

Browser: Chromium (via Playwright)

Process:

  1. Capture baseline accessibility state (patch not applied)
  2. Apply patch using git apply
  3. Clear Drupal cache and rescan
  4. Compare results and identify impact
  5. Generate reports in multiple formats
