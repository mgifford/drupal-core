# ISSUE-3087389: Media Library selection text is confusing for screen readers

**Original issue:** https://www.drupal.org/project/drupal/issues/3087389

## Plain-language summary

- When adding media through the Media Library modal, the text showing how many items are selected (e.g., "1 of 5 selected") is confusing for screen reader users and people with cognitive disabilities.
- The text reports the number of items selected out of the allowed maximum, but does not tell the user how many items are actually available to select in the grid.
- This can cause confusion if, for example, only 2 items are visible but the field allows 5, so the user hears "1 of 5 selected" but only finds 2 items to choose from.
- The grid of media items is not marked up as a list, so screen readers do not announce the number of items available.

## Local URL(s) to test

- Media Library modal: Open the media library from a field that allows media selection (e.g., add or edit content with a media field).
- Example: [http://drupal-core.ddev.site/node/add/article](http://drupal-core.ddev.site/node/add/article) (if an "Image" or "Media" field is present)

## How to quickly assess

1. Log in as an admin (`ddev drush uli` for one-time login).
2. Go to a content type with a media field (e.g., Article > Image/Media field).
3. Click "Add media" to open the Media Library modal.
4. Use a screen reader or inspect the text that reports selected items (e.g., "1 of 5 selected").
5. Check if the number of available items in the grid is announced or clear.
6. Assess if the selection text is understandable and not misleading.
